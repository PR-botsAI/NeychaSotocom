# Neycha Soto - Nail Artist Professional Website

## Repository Description
A specialized digital healthcare platform focused on nail restoration solutions for the Puerto Rican market, providing a comprehensive and culturally tailored management system. The platform delivers a seamless, patient-centric experience with advanced technological integrations.

## Core Technologies
- **Frontend**: React.js with TypeScript
- **UI Framework**: Radix UI with mobile-first responsive design  
- **Styling**: Tailwind CSS with dark theme
- **Database**: PostgreSQL with Drizzle ORM
- **Server**: Express.js with Vite
- **Localization**: Full Spanish language support
- **Performance**: Optimized web application with React Query
- **Accessibility**: WCAG compliant with ARIA support
- **SEO**: Advanced meta tags and schema markup for regional targeting

## Project Architecture

### Frontend Structure
- `/client/src/pages` - Main page components (Home, Onicoplastia, Contact)
- `/client/src/components` - Reusable UI components
- `/client/src/components/ui` - Shadcn/Radix UI components
- `/client/src/hooks` - Custom React hooks
- `/client/src/lib` - Utility functions and configurations
- `/client/src/data` - Static data and content

### Backend Structure
- `/server` - Express server with API routes
- `/db` - Database schema and migrations
- API endpoints for gallery, bookings, and content management

### Key Features
1. **Onicoplastia Service Showcase** - Interactive before/after comparisons
2. **Nail Condition Matcher** - Interactive tool for identifying nail conditions
3. **Booking Integration** - Direct integration with Booksy platform
4. **Gallery System** - Dynamic image gallery with categories
5. **Contact System** - WhatsApp integration for consultations
6. **Shop Promotion** - Integration with new shop at shop.neychasoto.com
